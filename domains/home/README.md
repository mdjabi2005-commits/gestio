# Module Home - Landing Page

**Dernière mise à jour** : 12 Février 2026

---

## 🎯 Vue d'Ensemble

Le module **Home** est une simple page d'atterrissage ("Landing Page") pour l'application. Elle sert de point d'entrée pour orienter l'utilisateur vers les différentes fonctionnalités (Ajout de transaction, Consultation, etc.).

---

## 📄 Fichiers

### Structure
```
domains/home/
├── __init__.py
├── pages/
│   └── home.py          # Page d'accueil simple
```

### `home.py`
Affiche un message de bienvenue et des raccourcis vers les actions principales (Ajouter, Voir Transactions).

---

## 📦 Dépendances
Aucune dépendance complexe (uniquement `streamlit`).
